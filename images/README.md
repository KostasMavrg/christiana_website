# Images Directory

Place your photos here:

1. **site-theme.jpg** - The main theme image (two profiles artwork) - This will be used in:
   - Navigation bar logo (circular, 60x60px)
   - Hero section main image
   - Subtle background theme throughout the site
   - Recommended size: 800x1000px or similar portrait/vertical ratio

2. **profile-photo.jpg** - Your main profile photo for the hero section (recommended size: 450x600px or similar portrait ratio)
3. **about-photo.jpg** - Your photo for the About section (recommended size: 350x450px or similar portrait ratio)

## Image Requirements:
- Format: JPG or PNG
- Recommended: High quality images
- The images will automatically resize to fit the design
- If images are not found, placeholder graphics will be displayed

## Site Theme Image:
The `site-theme.jpg` image will be used as:
- A circular logo in the navigation bar
- The main hero section image
- A subtle blurred background throughout the hero section
